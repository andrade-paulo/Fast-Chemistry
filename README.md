# Fast-Chemistry
 Chemistry calculator

Projeto open-source responsivo capaz de resolver operações químicas básicas. 

Operações:
 - Emissões de partículas alfa e beta
 - Ligações iônicas
 - Equação geral dos gases
 - Equação de Clayperon

Em desenvolvimento:
 - Balanceamento químico
 - Tabela periódica
